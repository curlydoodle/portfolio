## Run It Locally

To run it locally, enter commands down below to a powershell.

```bash
 # clone this repository
 git clone https://github.com/mustafaaakin/portfolio

 # go into the repository
 cd portfolio
```

```bash
 npm install # or yarn
 npm run dev # or yarn dev
```
